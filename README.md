## How to use

Download the files or Ctrl+C Ctrl+V the code.

### C:

`cc [filename].c` or `bcc [filename].c`\
`./a.out`

### Java:

**[filename] needs to be same as the name of the main class.**

`javac [filename].java`\
`java [filename]`

or

`java [filename].java` (Java 11+)
